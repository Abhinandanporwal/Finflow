# FinFlow 💰

<div align="center">

### Smart Finance & Transaction Management Platform

FinFlow is a modern finance management application designed to help users track transactions, manage expenses, analyze spending habits, and gain financial insights through an intuitive and interactive interface.

</div>

# ⚙️ Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/finflow.git
cd finflow
```

---

## 2️⃣ Install Dependencies

```bash
npm install
```

---

## 3️⃣ Setup Environment Variables

Create a `.env` file in the root directory.

```env
DATABASE_URL=

NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=

CLERK_SECRET_KEY=
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
```

---

## 4️⃣ Setup Prisma

```bash
npx prisma generate
npx prisma db push
```

---

## 5️⃣ Run the Development Server

```bash
npm run dev
```

Open in browser:

```bash
http://localhost:3000
```
